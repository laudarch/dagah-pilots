Instructions for getting the web interface to work:

1.  Make sure the bottle framwork for python is installed. installation instructions
    can be found at http://bottlepy.org/docs/dev/tutorial.html#installation.
    if you are on linux you should be able to use the following commands to install bottle
    
    sudo pip install bottle
    
2.  Edit /web-interface/index.py and change the line "run(host='192.168.203.128')" to match your ip address.

2.  run the following command, "python -m index.py" from the /web-interface directory.
    
3.  Visit 127.0.0.1/index.html
